# Kebijakan Privasi

Aplikasi Jadwal Ternak menghargai dan melindungi privasi pengguna. Aplikasi ini tidak mengumpulkan, menyimpan, atau membagikan data pribadi pengguna kepada pihak ketiga.

## Informasi yang Dikumpulkan
Aplikasi hanya menyimpan data yang dimasukkan oleh pengguna secara lokal di perangkat, seperti data ternak dan jadwal reproduksi. Data tersebut tidak dikirim ke server mana pun dan tidak dapat diakses oleh pihak lain.

## Keamanan Data
Seluruh data disimpan secara lokal menggunakan database internal perangkat (Room Database). Aplikasi dapat digunakan sepenuhnya tanpa koneksi internet.

## Perubahan Kebijakan
Kebijakan privasi ini dapat diperbarui sewaktu-waktu. Setiap perubahan akan diinformasikan melalui pembaruan aplikasi.
